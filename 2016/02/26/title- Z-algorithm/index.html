前言

相比于字符串排序，字符串匹配理解起来就没那么容易了，而且实现其实也比较麻烦。这里我整理下最近学的三个字符串匹配算法。但是如果将三个算法放一篇里无疑是太长了。今天先来看看Z-algorithm。

开篇

首先来定义需要解决的问题以及一些可能接下来常用的表示方式。

字串搜寻算法（String searching algorithms，又译字符串搜索算法）又称字串比对算法（string matching algorithms）是一种搜索算法，是字串算法中的一类，用以试图在一长字符串或文章中，找出其是否包含某一个或多个字符串，以及其位置.[1]

通常使用T来表现需要搜寻的文本字符串，使用P表示待匹配的字符串。比如：
T: This is a text string
P: is
那么P在T中出现，且出现次数为2。

虽然中文也能够进行字符串匹配，但是其算法完全不同，这里讨论的是英文字符串的匹配算法。
